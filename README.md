# ExampleCompanyApp

ExampleCompanyApp, ASP.NET Core kullanılarak geliştirilmiş, katmanlı mimari prensiplerine uygun bir Web API projesidir. Bu proje, sürdürülebilir, ölçeklenebilir ve temiz kod prensiplerine uygun bir backend mimarisi oluşturmayı amaçlamaktadır.

## 📌 Proje Amacı
Bu projenin amacı:
Katmanlı mimari (Layered Architecture) uygulamak,
Repository ve Service pattern kullanımı,
Temiz kod prensiplerine uygun backend geliştirmek,
Modern .NET backend geliştirme pratiği kazanmak.

## 🏗️ Kullanılan Teknolojiler
ASP.NET Core Web API
C#
Entity Framework Core
Repository Pattern
Service Layer Pattern
Caching Layer
SQL Server
Dependency Injection

## 🧩 Kullanılan Mimari Patternler

- Layered Architecture
- Repository Pattern
- Service Layer Pattern
- DTO Pattern
- AutoMapper (Object Mapping)
- Dependency Injection
- Caching

## 📂 Proje Mimarisi
Proje aşağıdaki katmanlardan oluşmaktadır:
🔹 ExampleCompanyApp.Api
API endpoint’lerini içerir
HTTP request/response işlemlerini yönetir
Controller sınıfları bu katmanda bulunur
🔹 ExampleCompanyApp.Core
Entity modelleri
Interface tanımları
Ortak veri yapıları
🔹 ExampleCompanyApp.Repository
Veritabanı işlemlerini yönetir
CRUD operasyonlarını içerir
Entity Framework Core kullanır
🔹 ExampleCompanyApp.Service
Business logic (iş kuralları) burada yer alır
Repository ile API arasında köprü görevi görür
🔹 ExampleCompanyApp.Caching
Performans optimizasyonu için caching işlemlerini içerir.

## 📌 Mimari Özellikler
Katmanlı mimari

Maintainable ve scalable yapı
Clean code yaklaşımı
